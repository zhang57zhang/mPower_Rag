2. ✅ **消息发送** - 发送消息，支持多轮对话
3. ✅ **对话历史** - 自动维护对话历史（最多 10 条）

### 架构设计

```
┌─────────────────────────────────────────────┐
│           前端 (Streamlit)                │
│  • 对话历史面板                           │
│  • 聊天界面                               │
│  • 消息输入和展示                         │
└──────────────┬────────────────────────────┘
               │ HTTP API
┌──────────────▼────────────────────────────┐
│         API 层 (FastAPI)                  │
│  • 5 个对话管理接口                      │
│  • 6 个数据模型                          │
└──────────────┬────────────────────────────┘
               │
┌──────────────▼────────────────────────────┐
│    对话管理器 (ConversationManager)       │
│  • 创建、获取、删除对话                   │
│  • 添加、获取消息                         │
│  • 维护上下文                             │
└──────────────┬────────────────────────────┘
               │
┌──────────────▼────────────────────────────┐
│       RAG 引擎 (RAGEngine)               │
│  • 加载对话历史                          │
│  • 格式化上下文                          │
│  • 生成答案                              │
│  • 保存消息                              │
└──────────────┬────────────────────────────┘
               │
┌──────────────▼────────────────────────────┐
│      向量存储 (VectorStore)               │
│  • 检索相关文档                          │
└───────────────────────────────────────────┘
```

### API 接口

| 方法 | 路径 | 功能 |
|------|------|------|

| GET | `/api/v1/conversations/{id}` | 获取对话详情 |
| POST | `/api/v1/conversations/{id}/messages` | 发送消息 |
| GET | `/api/v1/conversations` | 列出所有对话 |
