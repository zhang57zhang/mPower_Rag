# mPower_Rag 测试计划

## 1. 测试范围

### 1.1 测试目标
- 验证API接口功能正确性
- 验证文档上传、解析、存储功能
- 验证搜索和问答功能
- 验证边界条件和异常处理

### 1.2 测试环境
- Python 3.10+
- FastAPI 服务运行在 http://localhost:8000
- Streamlit 前端运行在 http://localhost:8501

## 5. 缺陷等级定义

| 等级 | 描述 |
|------|------|
| P0 - 致命 | 系统崩溃、数据丢失、核心功能完全不可用 |
| P1 - 严重 | 主要功能不可用，无临时解决方案 |
| P2 - 一般 | 功能可使用但有明显问题，有临时解决方案 |
| P3 - 轻微 | 界面问题、提示信息不准确等 |
