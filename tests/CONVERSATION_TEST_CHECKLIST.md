## 📝 测试步骤

### 1. 启动服务
```bash
scripts\start_all.bat
```

### 2. 验证服务启动
- [ ] Qdrant 已启动（检查 Docker 容器）
- [ ] FastAPI 已启动（检查终端输出）
- [ ] Streamlit 已启动（检查终端输出）

### 3. 手动测试（通过前端）

#### 3.1 访问前端
- [ ] 打开 http://localhost:8501
- [ ] 界面正常显示

#### 3.2 创建对话
- [ ] 点击 "➕ 新建对话"
- [ ] 对话 ID 显示在标题中
- [ ] 侧边栏显示新对话

#### 3.3 发送消息
- [ ] 输入问题（如："什么是车载测试？"）
- [ ] 点击 "🚀 发送"
- [ ] 等待答案生成
- [ ] 检查答案是否合理
- [ ] 检查是否显示源文档

#### 3.4 多轮对话
- [ ] 继续输入相关问题（如："测试流程有哪些？"）
- [ ] 检查答案是否参考了之前的对话
- [ ] 检查对话历史是否正确显示

#### 3.5 对话切换
- [ ] 点击 "➕ 新建对话" 创建第二个对话
- [ ] 在新对话中提问
- [ ] 侧边栏点击第一个对话
- [ ] 检查是否切换回第一个对话

#### 3.6 删除对话
- [ ] 选中一个对话
- [ ] 点击对话旁边的 🗑️ 按钮
- [ ] 确认对话已删除

### 4. 自动化测试（通过脚本）
```bash
python scripts\test_conversation.py
```
- [ ] 测试通过
- [ ] 没有报错

### 5. API 测试（可选）
- [ ] 测试 POST /api/v1/conversations
- [ ] 测试 GET /api/v1/conversations/{id}
- [ ] 测试 POST /api/v1/conversations/{id}/messages
- [ ] 测试 GET /api/v1/conversations
